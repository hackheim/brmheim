BrmHeim - LEDs and audio controlled tunnel

Welcome to API server of BrmHeim project, funded by NTNU (NO), TEKS (NO), Meta.Morf (NO), CIANT (CZ), done in cooperation of hackerspaces Brmlab, Prague (CZ) and HackHeim, Trondheim (NO)

What ? (Project description)

In tunnel below the road, next to Adresseavien, in the AdressaParken, we've added LED strips on the ceiling and connected them with audio/noise level detection, to visualise the environment and let people interact closely with the park itself.

Where ?

JSON read-only API available

  • Noise Level data gathering
    • Use /volume to get up to 1000 of volume samples, and list of volume measuring units
    • To get more specific data, from either single client or from all clients in specific time-span use URL templates below
      • /volume/CLIENT_ID
      • /volume/CLIENT_ID/DATETIME_FROM/DATETTIME_TO
      • /volume/CLIENT_ID/DATE_FROM/DATE_TO
      • /volume/CLIENT_ID/DATE_FROM/DATETIME_TO
      • /volume/CLIENT_ID/DATETIME_FROM/DATE_TO
      • /volume/DATETIME_FROM/DATETTIME_TO
      • /volume/DATE_FROM/DATE_TO
      • /volume/DATE_FROM/DATETIME_TO
      • /volume/DATETIME_FROM/DATE_TO
    • CLIENT_ID is id of client, currently there is only one, with CLIENT_ID=1
    • DATE_FROM/DATE_TO is in format YYYY-MM-DD (month and day is zero prefixed, year in full form), ie. 2016-04-16
    • DATETIME_FROM/DATETIME_TO is in format YYYY-MM-DD-HH-mm-ss (time is also zero prefixed, in 24-hour format), ie. 2016-04-16-13-30-00
